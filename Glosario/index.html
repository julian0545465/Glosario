Hipertexto

Lenguaje de programación que permite establecer vínculos entre diferentes bloques de información y moverse rápidamente entre ellos. El hipertexto fue integrado en la World Wide Web para crear referencias cruzadas entre las páginas disponibles en Internet y de esta manera facilitar el salto de una a otra
